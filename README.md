OpenWrite is a powerful Windows text editor designed for high school students writing college application essays. With rich-text formatting, inline comments, version tracking, and real-time word and sentence stats, OpenWrite helps students draft, review, and polish essays efficiently in a distraction-free, customizable environment.

## Features

__1. Rich Text Editing__

- Bold, italic, underline, font size adjustments, and paragraph alignment.
- Customizable line spacing, margins, and indentation for polished formatting.
- Intuitive toolbar and keyboard shortcuts for quick editing.

__2. Inline Comments__

- Highlight text and add comments directly in the editor.
- Supports multiple editors and collaborative feedback.
- Comments visually stand out with bold yellow highlights.

__3. Version Control__

- Save and restore multiple versions of a document.
- Snapshots include both text content and comments.
- Easily revert to previous drafts while tracking progress.

__4. Word & Sentence Statistics__

- Real-time tracking of word count, letter count, and sentence count.
- Helps students meet essay requirements and maintain readability.
- Dynamic updates as text is typed or edited.

__5. File Management__

- Open, save, and save-as support for `.wdoc` files.
- Export and import full documents including comments and versions.
- Simple workflow to keep essays organized locally.

__6. Print & Export__

- Print directly from the editor with proper page formatting.
- Maintains layout, margins, and line spacing for professional output.

## __Frequently Asked Questions__

### __1. Is OpenWrite free to use?__
Yes. __OpenWrite is free to use__, with all core editing and versioning features included at no cost.

---

### __2. What platforms does OpenWrite support?__
OpenWrite currently supports __Windows-based systems__ only.

---

### __3. Can OpenWrite track versions of my essay drafts?__
__Yes.__ OpenWrite includes built-in __version control__, letting you save, restore, and compare multiple drafts.

---

### __4. Can I collaborate with comments in OpenWrite?__
Yes. __Inline comments__ allow multiple editors to provide feedback while highlighting relevant text.

---

### __5. Is my essay data stored in the cloud?__
No. All documents, comments, and versions are stored __locally on your computer__ by default. No cloud storage is used.

---

### __6. How do I create a new document?__
Use the `New` menu option. OpenWrite ensures existing unsaved changes are prompted before starting a new document.

---

### __7. Does OpenWrite support printing essays?__
__Yes.__ Print directly from the editor, preserving page layout, margins, and line spacing.

---

### __8. How are updates handled?__
OpenWrite does not require online updates; new versions are provided on GitHub releases for manual download.

---

### __9. Where can I report bugs or request features?__
Please submit issues or feature requests via __GitHub Issues__ in the project repository.

## __Installation__

You can install OpenWrite using one of the following methods:

__1. Download the latest release__

1. Download the latest release from [Releases](https://github.com/FreeCoderExperience/OpenWrite/releases).
2. Download the ZIP file for the latest version.
3. Extract the ZIP and run `OpenWrite.exe`.

__2. Clone the repository with Git__

```bash
git clone https://github.com/FreeCoderExperience/OpenWrite.git
cd OpenWrite
```

__3. Download using cURL__

```bash
curl -LO https://github.com/FreeCoderExperience/OpenWrite/archive/refs/heads/main.zip
unzip main.zip
cd OpenWrite-main
```
